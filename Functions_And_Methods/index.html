Functions are defined as a block of code which performs its individual/specific tasks whenever called.

>>Functions parameters are like local variables of functions more precisely "Block Scope"

Syntax

1. function func(parameter){ return statement; }

2. const func =(prompt)=>{ return statement; }

>>Higher Order Functions/Methods : Such functions which either takes another functions as a parameter "OR" return another functions.

SOME MORE ARRAY METHODS

1. forEach() Method
Returns every element of the particular array
example: arr.forEach((val)=>{ return val*val; })
2. map() Method
Creates a new array of elements according to the function
example: let new_array = arr.map((val)=>{ return val%10; })
3. filter() Method
Creates a new array of elements that give true for a condition/filter
example: let new_array = arr.filter((val)=>{ return val%10==0; })
4. reduce() Method
Performs some operations and reduces the array to the single value. It returns that single value
example (find the smallest element in an array): console.log(array.reduce((prev,curr)=>{ return prev < curr ? curr : prev; }))
